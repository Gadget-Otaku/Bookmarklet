# Shortcut Panel

Webブラウジングを効率化する、カスタマイズ可能なフローティングショートカットランチャー拡張機能です。
あらゆるWebサイト上にオーバーレイ表示されるパネルから、よく使うサイトやブックマークレットへ瞬時にアクセスできます。Shadow DOM技術を採用しており、閲覧中のサイトのデザインを崩すことなく動作します。

## 🌟 主な機能

* **フローティングパネル**:
    * **オーバーレイ表示**: サイトの最前面に半透明（グラスモーフィズム風）のパネルを表示。ドラッグして好きな位置に移動可能です。
    * **フォルダ管理**: ショートカットをフォルダにまとめて整理可能。階層構造ですっきり管理できます。
    * **外観カスタマイズ**: グリッドの列・行数、アイコンサイズ、全体スケールなどを細かく調整可能。
    * **自動テーマ適応**: 背景色に応じて文字色（白/黒）を自動または手動で切り替え、視認性を確保します。
* **強力なトリガー設定**:
    * **表示ルール**: 「すべてのサイトで表示」「特定のサイトのみ表示」「特定のサイトを除外」など、柔軟なルール設定が可能。
    * **トリガーボタン**: パネルを呼び出すためのフローティングボタンを画面端に設置可能（透過モード、スクロール時非表示など対応）。
    * **キーボードショートカット**: 任意のキーを設定し、キー入力だけでパネルを呼び出し可能。
* **高機能エディタ**:
    * **GUI編集**: パネル上から設定画面を開き、直感的にショートカットの追加・編集・並べ替えが可能。
    * **アイコン自動取得**: URLを入力するだけで、GoogleのFavicon APIを利用してアイコンを自動設定。
    * **バックアップ**: 設定内容はJSON形式でインポート・エクスポート可能。ブックマークレット形式のコード読み込みにも対応しています。

## 📥 インストール方法

お使いのブラウザに合わせてファイルをダウンロード・インストールしてください。

### 🟣 Chromium系ブラウザをお使いの方
**対象**: Google Chrome, Brave, Microsoft Edge, Vivaldi, Opera など

1.  [/dist](https://github.com/Gadget-Otaku/Bookmarklet/tree/main/%E6%8B%A1%E5%BC%B5%E6%A9%9F%E8%83%BD/Shortcut%20Panel/dist)から **`.zip` ファイル** をダウンロードし、解凍（展開）してください。
2.  ブラウザの拡張機能管理ページを開きます。
    * Chrome/Brave/Edge: `chrome://extensions`
3.  右上の **「デベロッパーモード」** をオンにします。
4.  **「パッケージ化されていない拡張機能を読み込む」** (Load unpacked) をクリックします。
5.  解凍したフォルダ（`manifest.json` が入っているフォルダ）を選択するとインストールされます。

### 🦊 Firefox系ブラウザをお使いの方
**対象**: Firefox Nightly, LibreWolf, Zen Browser, Floorp, Iceraven など

1.  [/dist](https://github.com/Gadget-Otaku/Bookmarklet/tree/main/%E6%8B%A1%E5%BC%B5%E6%A9%9F%E8%83%BD/Shortcut%20Panel/dist)から **`.xpi` ファイル** をダウンロードしてください。
2.  `about:config`を開き、`xpinstall.signatures.required`と入力し、値を`false`に変更します。(初回のみ)
3.  ブラウザの設定メニューから「アドオンとテーマ (`about:addons`)」を開きます。
4.  歯車アイコン ⚙️ をクリックし、**「ファイルからアドオンをインストール」** を選択します。
5.  ダウンロードした `.xpi` ファイルを選択してインストールしてください。

> **注意**: 通常のFirefox安定版では、未署名のアドオンを永続的にインストールできません。開発者用エディション (Developer Edition) や Nightly、または Floorp などの派生ブラウザを使用することを推奨します。

## 📱 使い方

### 基本操作
1.  **パネルを開く**:
    * 画面端（初期設定では左上付近など）にあるトリガーボタンをクリックするか、設定したキーボードショートカットを押します。
2.  **ショートカットの使用**:
    * アイコンをクリックすると、設定されたURLへ移動します。
    * **新しいタブで開く**: パネル右上の「+」ボタンをクリックして有効（ハイライト状態）にすると、ショートカットが常に新しいタブで開くようになります。
    * **ダークモード切替**: パネル右上の「◯/●」ボタンで、パネルの文字色（白/黒）を手動で切り替えられます。
3.  **パネルの移動**:
    * パネルの余白部分をドラッグ＆ドロップすることで、画面上の好きな位置に配置できます。

### 設定・編集 (Editor)
パネル右上の **⚙️ (歯車)** アイコンをクリックすると、詳細なエディタ画面が開きます。

* **追加**: 現在開いているページのURLとタイトルを自動取得し、即座にパネルに追加できます。
* **ショートカット編集**: ツリービュー形式でアイテムの並べ替え、フォルダ化、アイコンURLの変更が可能です。
* **パネル編集 (表示ルール)**:
    * **サイト設定**: どのURLでパネルを有効にするか（前方一致/完全一致）。
    * **ボタン設定**: トリガーボタンの表示位置、色、透過度、カスタムテキストなどを設定。
    * **キーボード設定**: パネルを開閉するホットキーを割り当て。
* **インポート/エクスポート**: 設定データをJSONテキストとしてバックアップしたり、別のブラウザへ移行したりできます。

問題番号: 1.3.9
分野: Simple random sample / Random number table / Pearson MyLab

問題の概要:
A community college employs 85 full-time faculty members.
Faculty are numbered from 1 to 85.
A simple random sample of 9 faculty members is needed.
Using the provided random number table, start at the digit in row 3, column 6 and proceed downward.

乱数表:
Row,01-05,06-10,11-15,16-20
01,28795,18552,54468,91120
02,34966,64774,67322,68548
03,61745,87647,54807,65078
04,57937,49249,33859,01689
05,82243,69948,65945,17031
06,01993,91423,37040,13563

確定正解:
49, 69, 55, 77, 64, 24, 42, 25, 46

重要:
この問題では「row 3, column 6 の digit は 8 だから、8 から1桁ずつ縦に読んで後から2桁に区切る」と考えると不正解になる。
誤答例:
84, 69, 79, 57, 62, 44, 42, 24, 83

Pearson / MyLab 系での正しい考え方:
1. 母集団は 85 人なので、番号は 01〜85 の2桁として扱う。
2. row 3, column 6 は開始地点の digit を示すだけであり、読み取り単位は母集団に合わせて2桁にする。
3. したがって、row 3 column 6 と row 3 column 7 を合わせて、最初の2桁候補は 87 になる。
4. “proceeding downward” は、1桁だけの列を下へ読むという意味ではなく、必要な桁数ぶんの幅、今回は2桁幅のレーンを下へ読むという意味で扱う。
5. 最下行まで読んだら、次の未使用の列ペアの一番上に移る。
6. 01〜85 の範囲に入る番号だけを採用する。
7. 86以上、00、重複番号はスキップする。
8. 有効な番号が9個そろうまで続ける。

計算手順:

列 6-7 の2桁レーンから開始する。
row 3: 87 → 85を超えるのでスキップ
row 4: 49 → 採用 1
row 5: 69 → 採用 2
row 6: 91 → 85を超えるのでスキップ

次に、最下行まで到達したので、次の未使用の列ペアである列 8-9 の一番上へ移る。
row 1: 55 → 採用 3
row 2: 77 → 採用 4
row 3: 64 → 採用 5
row 4: 24 → 採用 6
row 5: 94 → 85を超えるのでスキップ
row 6: 42 → 採用 7

次に、列 10-11 の2桁レーンへ移る。
5桁ブロックの境界は見やすさのための区切りなので、column 10 と column 11 をまたいで読んでよい。
row 1: column 10 = 2, column 11 = 5 → 25 → 採用 8
row 2: column 10 = 4, column 11 = 6 → 46 → 採用 9

最終的な採用番号:
49, 69, 55, 77, 64, 24, 42, 25, 46

AI向け注意:
この問題では wording の “points to the digit” を「1桁ずつ読む」と解釈しすぎないこと。
Pearson / MyLab の乱数表問題では、母集団の最大番号に必要な桁数を先に決める。
今回は 85 人なので2桁。
開始 digit は2桁レーンの左端、つまり十の位の開始位置として使う。
そのため row 3 column 6 から始まる最初の候補は 84 ではなく 87。
“proceeding downward” は、その2桁レーンを下へ進むという意味。
類題で母集団が 1〜99 なら2桁、1〜999 なら3桁のレーンを作って、指定方向に読む。

問題の概要:
A scatter diagram is given with one of the points drawn in blue (large point).
The red dashed line is the least-squares regression line with the blue point excluded.
The blue solid line is the least-squares regression line with the blue point included.
Question: On the basis of these graphs, do you think the point in blue is influential?

選択肢:
・No, because the point significantly affects the least-squares regression line.
・No, because the point does not significantly affect the least-squares regression line.
・Yes, because the point does not significantly affect the least-squares regression line.
・Yes, because the point significantly affects the least-squares regression line.

確定正解:
No, because the point does not significantly affect the least-squares regression line.

誤答例:
Yes, because the point significantly affects the least-squares regression line.

問題番号: 未確認
分野: Least-squares regression line / Regression / Pearson MyLab

問題の概要:
The time it takes for a planet to complete its orbit around a particular star is called the planet's sidereal year.
The data show the distances of the planets from a particular star and their sidereal years.
Compute the least-squares regression line.

データ:
Distance from the Star, x (millions of miles): 36, 67, 93, 142, 483, 887, 1785, 2797, 3675
Sidereal Year, y: 0.24, 0.62, 1.00, 1.86, 11.8, 29.5, 82.0, 163.0, 248.0

確認:
x は小さい順で 36, 67, 93, 142, 483, 887, 1785, 2797, 3675
n = 9

確定正解:
y_hat = 0.065x + (-12.6)

正確な計算値:
slope = 0.06537083433052136
intercept = -12.60004045596059

丸め:
slope は小数第3位までなので 0.065
intercept は小数第1位までなので -12.6

誤答例:
y_hat = 0.067x + (-16.3)

重要:
この問題では中間値を丸めず、全9点をそのまま使って最小二乗回帰直線を計算する。
0.067, -16.3 は、与えられた9点全体から計算した least-squares regression line と一致しない。
実際には、(93, 1.00) と (142, 1.86) の2点を落として7点だけで計算すると、slope = 0.0667203064..., intercept = -16.2897974... となり、丸め後の誤答 0.067, -16.3 とほぼ一致する。
したがって、この誤答は単なる丸めミスではなく、データ点の抜け落ちによる可能性が高い。

AI向け注意:
統計・回帰問題では、必ず Python / numpy / statistics 等でデータを配列として定義し、n を確認してから計算する。
説明文に9点を書いていても、内部計算で配列に9点すべて入っているとは限らないので、計算前に n=9 を確認する。
回帰式の入力欄が y_hat = ____ x + ( ____ ) の形なら、切片が負の場合は括弧内に -12.6 を入れる。
